# 🌍 BharatYatra - India Tours

BharatYatra is a custom tour curation web platform designed to deliver authentic, safe, and budget-friendly domestic travel packages across India. The project features an interactive landing page with a dynamic pricing slider, detailed location guides, an online hotel and room booking form, and a direct contact channel for customer support. Built entirely with HTML, CSS, and lightweight JavaScript, it provides a fully responsive UI for travelers exploring regional destinations.

---

## 🚀 Getting Started

Follow these steps to download the repository and run the website locally on your machine.

### 1. Clone the Repository
Copy and paste the following command into your terminal to clone the project:
git clone [https://github.com/Anish4104/BharatYatra.git](https://github.com/Anish4104/BharatYatra.git)

###2. Navigate to the Directory
Change your current working directory to the project folder:
Bash
cd BharatYatra

###3. Launch the Application
To view the live website, simply open the main file in your preferred web browser:

Bash
open index.html
(Alternatively, you can just double-click the index.html file inside the project folder or launch it using an IDE extension like Live Server.)
